# Arduino 4bit adder
<img width="1601" height="955" alt="2026-02-07_10-28" src="https://github.com/user-attachments/assets/35e4f9f8-479d-4d99-ac1d-171606b102bf" />

Simple Arduino calculator that can add two 4bit numbers together. A simulation can be found on [Tinkercad](https://www.tinkercad.com/things/6ArvLugKSeX-arduino-4bit-adder?sharecode=Qj4UFxbOs13o_4GKma6KRRp_bLDndrn4tgFKTXqORo8).

### Controls
- First button - clear both numbers, reset index
- Second button - add both numbers together and store the result in the green LEDs, reset index
- Third button - set a bit on in the white LEDs and advance the index
- Fourth button - set a bit off in the white LEDs and advance the index
